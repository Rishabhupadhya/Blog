## Microservices Overview

Microservices architecture is an approach where a single application is developed as a suite of small services.

### Benefits

- Independent deployment
- Technology diversity
- Fault isolation

## Design Patterns

### API Gateway Pattern

Provides a single entry point for clients.

### Service Discovery

Dynamic service location and load balancing.

### Circuit Breaker

Preventing cascading failures in distributed systems.
