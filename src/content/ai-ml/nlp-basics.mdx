## NLP Overview

Natural Language Processing enables computers to understand human language.

### Key Applications

- Text classification
- Sentiment analysis
- Machine translation

## Core Concepts

### Tokenization

Breaking text into smaller units.

### Word Embeddings

Representing words as vectors (Word2Vec, GloVe).

### Transformers

Modern NLP architecture (BERT, GPT).
